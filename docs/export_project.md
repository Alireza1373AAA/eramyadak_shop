# خروجی گرفتن کامل از پروژه

برای دریافت یک آرشیو کامل از سورس‌کد پروژه می‌توانید از اسکریپت `scripts/export_project.sh` استفاده کنید. این اسکریپت بر اساس نسخه‌ای که در گیت وجود دارد خروجی می‌گیرد؛ بنابراین فقط فایل‌های دنبال‌شده توسط گیت در آرشیو قرار می‌گیرند.

## پیش‌نیاز
- نصب بودن Git
- قرار داشتن در ریشه مخزن (پوشه‌ای که فایل `.git` در آن وجود دارد)

## نحوه استفاده سریع
```bash
./scripts/export_project.sh
```

بعد از اجرای دستور بالا، فایل زیپ‌شده در مسیر `dist/eramyadak_shop-HEAD.zip` ساخته می‌شود. اگر برنچ فعلی نام متفاوتی داشته باشد، همان نام در فایل خروجی استفاده خواهد شد.

## گزینه‌های قابل تنظیم
- `--output <مسیر>`: مسیر و نام دلخواه فایل خروجی. مثال: `./scripts/export_project.sh --output /tmp/project.zip`
- `--format zip|tar`: فرمت خروجی. به‌صورت پیش‌فرض `zip` است، ولی می‌توانید `tar` را نیز انتخاب کنید.
- `--ref <ref>`: مرجع گیت موردنظر برای خروجی گرفتن (مثلاً `main` یا `v1.0.0`). به‌صورت پیش‌فرض `HEAD` است.

## مثال‌ها
```bash
# خروجی گرفتن از برنچ main با فرمت tar
./scripts/export_project.sh --ref main --format tar

# ذخیره خروجی در پوشه دلخواه
./scripts/export_project.sh --output ~/Desktop/eramyadak.tar
```

## نکات
- اگر فایل یا پوشه‌ای خارج از کنترل گیت دارید (مثلاً فایل‌های موقتی)، در آرشیو قرار نمی‌گیرند. در صورت نیاز، آن‌ها را قبل از اجرای اسکریپت در گیت اضافه و کامیت کنید.
- می‌توانید پس از تولید آرشیو آن را از مسیر `dist/` بردارید و برای ارسال به افراد دیگر یا بارگذاری در سرویس‌های مختلف استفاده کنید.
